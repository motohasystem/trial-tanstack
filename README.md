# TanStack Learning Playground 🚀

TanStackの主要なライブラリを学習するためのインタラクティブなプレイグラウンドアプリケーションです。

## 📚 含まれる機能

このアプリケーションでは、以下のTanStackライブラリの機能を学習できます：

### 1. React Query 🔄
- **基本的なクエリ**: データフェッチングとキャッシング
- **依存クエリ**: パラメータに基づく自動再フェッチ
- **ミューテーション**: データの作成・更新・削除
- **無限クエリ**: 無限スクロール実装

### 2. React Table 📊
- **基本的なテーブル**: シンプルなテーブル表示
- **ソート機能**: カラムによるソート
- **フィルタリング**: データの検索とフィルタリング
- **ページネーション**: 大量データのページ分割

### 3. React Virtual 📜
- **基本的な仮想リスト**: 10,000件の効率的なレンダリング
- **動的サイズのリスト**: 可変高さの要素の仮想化
- **水平方向の仮想リスト**: 横スクロールの仮想化
- **仮想グリッド**: 行と列の両方の仮想化

### 4. React Form 📝
- **基本的なフォーム**: シンプルなフォーム管理
- **バリデーション**: リアルタイムバリデーション
- **フィールド配列**: 動的なフィールドの追加・削除
- **条件付きフィールド**: 条件に応じたフィールド表示

## 🚀 セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## 📖 使い方

1. 左側のナビゲーションから学習したい機能を選択
2. 各ページで実際に動作するデモを確認
3. 「コードを表示」をクリックしてサンプルコードを参照
4. パラメータを変更して動作を試す

## 🛠 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - 型安全性
- **Vite** - 高速なビルドツール
- **TanStack Query** - サーバー状態管理
- **TanStack Table** - テーブル管理
- **TanStack Virtual** - 仮想化
- **TanStack Form** - フォーム管理
- **React Router** - ルーティング

## 🎯 学習のポイント

各プレイグラウンドページでは、以下を学べます：

- **実際の動作**: ライブデモで機能を体験
- **コードサンプル**: 実装方法を確認
- **パラメータ調整**: インタラクティブに動作を変更
- **ベストプラクティス**: TanStackの推奨される使用方法
